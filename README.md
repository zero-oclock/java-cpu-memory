# java-cpu-memery
- A script that can quickly locate CPU and memory problems on the Java line（一个可以快速定位Java线上CPU和内存问题的脚本）

# 即使CPU占用分析
1. 赋予脚本执行权限  
   chmod +x show_maximum_cpu.sh
2. 运行CPU分析脚本  
```
sh show_maximum_cpu.sh
```
